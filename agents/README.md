# Banco de Agentes de Trading

Este directorio almacena los agentes entrenados organizados por generación.

## Estructura

```
agents/
├── generation_1/
│   ├── AG-2023-001.pkl
│   └── metrics.json
└── generation_2/
    ├── AG-2023-002.pkl
    └── metrics.json
```

## Métricas

Cada agente almacena:
- Profit Factor
- Número de trades
- Máximo drawdown
- Sharpe ratio
- Indicadores utilizados
